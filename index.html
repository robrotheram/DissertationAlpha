Welcome to My Dissertation.

The title of my dissertation is to digitise the campus which is a very broad title. The purposed system is to build a piece of software that is able to render the large building of the campus and include some navigation to help a person navigate the campus. The objects are created in SFM (Structures from motion) software that is able to build a point cloud
